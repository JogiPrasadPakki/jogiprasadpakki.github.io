android-native-core.Settings

Description

With this class you can open almost every android Settings screens

Summary:

This class consist 83 android settings packages this dificult to describe all this constents in this page so read android developer article about this settings packages Android developer documentation.

Public methods
voidOpen(string id)

Pass settings package id string to open releated settings screen.

Example
    
using android-native-core;
class exampe{
public void openSettings()
{ 
//this method open wifi settings
Settings settings = new Settings();
settings.Open(Settings.ACTION_WIFI_SETTINGS );
}
}
